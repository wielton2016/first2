Image

Image(string file [, float x [, float y [, float w [, float h [, string type [, mixed link]]]]]])

Opis

Wstawia obrazek. Rozmiar obrazka który będzie na stronie może być wyspecyfikowany przez:
  • sprecyzowaną szerokość i wysokość (wyrażoną w jednostkach użytkownika lub DPI)
  • jedną sprecyzowaną wartość. Pozostałe wartości będą obliczone automatycznie tak, by trzymać oryginalny współczynnik proporcji
  • bez sprecyzowanego wymiaru, w tym przypadku obrazek jest wstawiany z rozdzielczością 96 dpi
Wspierane fotmaty to JPEG, PNG i GIF. Biblioteka GD jest wymagana dla GIF.

Dla JPEGów, wszystkie skale są dozwolone:
  • skala szarości
  • RGB (24 bits)
  • CMYK (32 bits)
Dla obrazków PNG, dozwolone są:
  • skala szarości na co najwyżej 8 bitów (256 poziomów)
  • kolory indeksowane
  • RGB (24 bits)
Dla obrazków GIF: w przypadku animowanego GIFu, jedynie pierwsza ramka będzie wykorzystana.

Wspierana jest przeźroczystość.

Format może to specyfikować bezpośrednio lub może to wynikać z rozszerzenia pliku.

Możliwe jest umieszczenie linku na obrazku.

Uwaga: jeśli obrazek jest używany kilka razy, jedynie jedna kopia jest osadzona w pliku pdf.

Parametry

file
Ścieżka lub url do obrazka.
x
Odcięta lewego górnego-rogu. Jeśli nie jest określona lub równa null, aktualna odcięta jest używana.
y
Rzędna lewego-górnego rogu. Jeśli nie jest wyspecyfikowana lub równa null, aktualna rzędna jest używana; ponadto podział strony jest przeprowadzany wcześniej, jeśli to niezbędne (w przypadku gdy automatyczne łamanie strony jest włączone) i, po wywołaniu, aktualna rzędna jest przenoszona na spód obrazka.
w
Szerokość obrazka na stronie. Są trzy możliwe przypadki:
  • Jeśli ta wartość jest dodatnia, oznacza to że szerokość jest w jednostkach użytkownika
  • Jeśli ta wartość jest ujemna, wartość bezwzględna reprezentuje rozdzielczość poziomą w DPI
  • Jeśli nie jest określona lub równa 0, jest automatycznie obliczana
h
Wysokość obrazka na stronie. Są trzy możliwe przypadki:
  • Jeśli ta wartość jest dodatnia, reprezentuje ona wysokość w jednostkach użytkownika
  • Jeśli ta wartość jest ujemna, wartość bezwzględna reprezentuje rozdzielczość pionową w DPI
  • Jeśli nie jest określona lub równa 0, jest automatycznie obliczana
type
Format obrazka. Możliwe wartości to (rozróżnia małe i wielkie litery): JPG, JPEG, PNG i GIF. Jeśli nie jest określony, typy obrazka jest określany na podstawie rozszerzenia pliku.
link
URL lub identyfikator zwracany przez AddLink().

Przykład

// Umieść logo z pliku w lewym górnym rogu z rozdzielczością 300 DPI (oddalone 10 pikseli od lewej strony i 10 pikseli od góry strony)
$pdf->Image('logo.png',10,10,-300);
// Umieść obrazek bezpośrednio z linku www (dynamicznie)
$pdf->Image('http://chart.googleapis.com/chart?cht=p3&chd=t:60,40&chs=250x100&chl=Hello|World',60,30,90,0,'PNG');
