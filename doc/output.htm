Output

string Output([string dest [, string name [, boolean isUTF8]]])

Opis

Wysyła dokument do określonego przeznaczenia: przeglądarki, pliku lub łańcuchu. W przypadku przeglądarki, może być wykorzystany plugin (jeśli jest obecny) lub opcja pobierania (Zapisz jako) może być wymuszona.
Metoda ta wpierw wywołuje Close() jeśli jest to niezbędne do zamknięcia dokumentu.

Parametry

dest
Przeznaczenie, gdzie dokument ma być wysłany. Może to być jedna z następujących wartości:
  • I: wysłanie i otwarcie w przeglądarce. Wtyczka programu pdf jest używana, jeśli jest dostępna.
  • D: wysłanie do przeglądarki i wymuszenie pobierania pliku z nazwą określoną przez name.
  • F: zapis do lokalnego pliku z nazwą określoną w name (może zawierać ścieżkę).
  • S: zwraca nazwę dokumentu jako łańcuch.
Domyslna wartoscia jest I.
name
Nazwa pliku. Jest ona ignorowana w przypadku typu wyjścia S.
Domyślną wartością jest doc.pdf.
isUTF8
Wskazuje czy name jest zakodowana w ISO-8859-1 (false) lub w UTF-8 (true).
Domyslna wartoscia jest false.
