SetLink

SetLink(int link [, float y [, int page]])

Opis

Definiuje stronę i pozycję do której ma wskazywać link.

Parametry

link
Identyfikator linku zwrócony przez AddLink().
y
Rzędna obiektu przeznaczenia; -1 wskazuje aktualną pozycję. Domyślną wartością jest 0 (góra strony).
page
Numer strony przeznaczenia; -1 wskazuje aktualną stronę. Jest to domyślna wartość.
