SetTextColor

SetTextColor(int r [, int g, int b])

Opis

Definiuje kolor używany dla tekstu. Może być one wyrażony w skali RGB lub skali szarości. Metoda ta może być wywoływana przed stworzeniem pierwszej strony i wartość ta jest przechowywana ze strony do strony.

Parametry

r
Jeżeli składowe g i b są podane, jest to składowa czerwona; jeżeli nie, wskazuje poziom szarości. Wartość pomiędzy 0 a 255.
g
Składowa zielona (pomiędzy 0 a 255).
b
Składowa niebieska (pomiędzy 0 a 255).
