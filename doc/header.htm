Header

Header()

Opis

Metoda ta jest używana do przedstawienia nagłówka strony. Jest automatycznie wywoływana przez AddPage() i nie powinna być wywoływana bezpośrednio przez aplikację. Implementacja w FPDF is pusta, więc musisz rozszerzyć ją i zastępić tą metodę, jeśli chcesz własnego przetwarzania.

Przykład

class PDF extends FPDF
{
function Header()
{
    // wybranie czcionki Arial bold 15
    $this->SetFont('Arial','B',15);
    // przesunięcie na prawo
    $this->Cell(80);
    // tytuł w ramce
    $this->Cell(30,10,'Title',1,0,'C');
    // przełamanie linii
    $this->Ln(20);
}
}
