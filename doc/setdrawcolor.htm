SetDrawColor

SetDrawColor(int r [, int g, int b])

Opis

Definiuje kolor używany dla wszystkich operacji malowania (linie, prostokąty, obramwania komórek). Może być wyrażony w skali RGB lub skali szarości. Metoda ta może być wywołana przed stworzeniem pierwszej strony a wartość jest przekazywana ze strony do strony.

Parametry

r
Jeżeli g i b są podane, składowa czerwona; jeśli nie, poziom wskazuje poziom szarości. Wartość pomiędzy 0 a 255.
g
Składowa zielona (pomiędzy 0 a 255).
b
Składowa niebieska (pomiędzy 0 a 255).
