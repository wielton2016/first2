Rect

Rect(float x, float y, float w, float h [, string style])

Opis

Rysuje prostokąt. Może być on narysowany (tylko obramowanie), wypełniony (bez ramki) lub oba.

Parametry

x
Odcięta lewego-górnego rogu prostokąta.
y
Rzędna lewego-górnego rogu prostokąta.
w
Szerokość.
h
Wysokość.
style
Styl kreślenia. Możliwe wartości to:
  • D lub puste: rysuj. Jest to wartość domyślna.
  • F: wypełnij
  • DF lub FD: wypełnij i rysuj.
