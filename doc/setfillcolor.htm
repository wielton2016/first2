SetFillColor

SetFillColor(int r [, int g, int b])

Opis

Definiuje kolor używany do operacji wypełnienia (wypełnienie prostokąta i tło komórek). Może być wyrażony w skali RGB lub skali szarości. Metoda ta może być wywołana przed stworzeniem pierwszej strony a wartość jest przekazywana ze strony do strony.

Parametry

r
Jeśli g i b są podane, jest to składowa czerwona; jeśli nie, wskazuje poziom szarości. Wartość pomiędzy 0 a 255.
g
Składowa zielona (pomiędzy 0 a 255).
b
Składowa niebieska (pomiędzy 0 a 255).
