Footer

Footer()

Opis

Ta metoda jest używana do umieszczania stopki strony. To jest automatycznie wywoływane przez AddPage() i Close() i nie powinna być wywoływana bezpośrednio przez aplikację. Implementacja w FPDF jest pusta, więc musisz nadpisać tą metodę jeśli chcesz wstawić swoją stopkę.

Przykład

class PDF extends FPDF
{
function Footer()
{
    // Idź do 1.5 cm od spodu
    $this->SetY(-15);
    // Wybierz czcionkę Arial italic 8
    $this->SetFont('Arial','I',8);
    // Drukuj centralnie numer strony
    $this->Cell(0,10,'Page '.$this->PageNo(),0,0,'C');
}
}
