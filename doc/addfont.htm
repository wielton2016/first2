AddFont

AddFont(string family [, string style [, string file]])

Opis

Importuje czcionkę TrueType, OpenType lub Type1 i czyni ją dostępną. Jest to niezbędne do generowania pliku definicji czcionki wpierw używając właściwości makefont.php.
Plik definicji (i plik z czcionką kiedy jest osadzona) musi być obecny w katalogu z czcionką. Jeśli go to nie ma, generowany jest błąd: "Could not include font definition file".

Parametry

family
Rodzina czcionek. Nazwa może być wybrana dowolnie. Jeśli to jest standardowa nazwa czcionki, to będzie zastępować odpowiednie czcionki.
style
Styk czcionki. Możliwe wartości to (rozróżnia duże i małe litery):
  • puste: normalna czcionka
  • B: pogrubiona
  • I: pochylona
  • BI lub IB: pogrubiona i pochylona
Wartością domyślną jest normalna czcionka.
file
Plik definicji czcionki.
Domyślnie, nazwa jest tworzona z rodziny czcionki i stylu, małymi literami bez spacji.

Przykład

$pdf->AddFont('Comic','I');
jest równoznaczne z: 
$pdf->AddFont('Comic','I','comici.php');
