Write

Write(float h, string txt [, mixed link])

Opis

Metoda ta drukuje tekst od aktualnej pozycji. Kiedy prawy margines jest osiągany (lub znak \n jest spotykany) następuje złamanie linii i tekst jest kontynuowany od lewego marginesu. Przy zakończeniu tej metody, aktualna pozycja jest na lewo ale od końca aktualnego tekstu.
Możliwe jest umieszczenie linku w tekście.

Parametry

h
Wysokość linii.
txt
Łańcuch do wydrukowania.
link
URL lub identyfikator zwrócony przez AddLink().

Przykład

// zacznij czcionką normalną
$pdf->SetFont('Arial','',14);
$pdf->Write(5,'Visit ');
// następnie daj niebieski podkreślony link
$pdf->SetTextColor(0,0,255);
$pdf->SetFont('','U');
$pdf->Write(5,'www.fpdf.org','http://www.fpdf.org');
