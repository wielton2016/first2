__construct

__construct([string orientation [, string unit [, mixed size]]])

Opis

Klasa konstruktora. Pozwala ustawić format strony, orientację i jednostki używane we wszystkich metodach (z wyjątkiem rozmiaru czcionek).

Parametry

orientation
Domyślna orientacja strony. Możliwe wartości to (z uwzględnieniem wielkości liter):
  • P lub Portrait
  • L lub Landscape
Domyślną wartością jest P.
unit
Jednostka użytkownika. Możliwe wartości to:
  • pt: punkt
  • mm: milimetr
  • cm: centymetr
  • in: cal
Punkt równa się 1/72 cala, czyli około 0.35 mm (cal ma 2.54 cm). Ta jednostka jest bardzo powszechna w poligrafii; rozmiary czcionek są wyrażone w tych jednostkach.

Domyślną wartością jest mm.
size
Format używanych stron. Może przyjmować jedną z następujących wartości (z uwzględnieniem wielkości liter):
  • A3
  • A4
  • A5
  • Letter
  • Legal
lub tablicę zawierającą szerokość i wysokość (wyrażone w jednostkach danych przez unit).

Domyślną wartością jest A4.

Przykład

Przykład z wybranym formatem strony 100x150 mm: 
$pdf = new FPDF('P','mm',array(100,150));
